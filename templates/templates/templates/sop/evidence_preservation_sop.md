# Evidence Preservation SOP (Basic)

## Purpose
Preserve evidence integrity so findings are defensible.

## Rules
1. Minimize handling.
2. Document every transfer.
3. Hash before and after imaging/copying.
4. Store evidence securely.
5. Work from copies whenever possible.

## Procedure
1. Intake and label the evidence item.
2. Complete the Evidence Intake Form.
3. Start the Chain of Custody log immediately.
4. Create a forensic image/copy using approved tools.
5. Generate hashes for the original and the copy.
6. Verify hashes match and document results.
7. Store original evidence and work only from the verified copy.

## Preventative measures
- Use tamper-evident bags when possible
- Keep consistent naming for case IDs and evidence IDs
- Maintain log retention for related systems
