# Forensics Evidence Starter Kit

This repository contains simple templates and procedures for evidence intake, chain of custody, and hashing documentation.

## Contents
- templates/
  - Evidence intake form
  - Chain of custody log
  - Hashing checklist
- sop/
  - Evidence preservation SOP

## How to use
1. Copy the templates into your case folder.
2. Fill them out during intake and processing.
3. Save completed versions with the case ID in the filename.

## Notes
This is for training and documentation practice. Do not include real personal data in this repo.
